# Fix NameError in `_ocr_only_fallback`

## Problem
`backend/services/vlm_structure_mapper.py:851` — `NameError: name 'text' is not defined`

In the `_ocr_only_fallback` method, the loop iterates over `sorted_regions` but never extracts `text` from each `region` dict before using it.

## Fix
In `_ocr_only_fallback` (line 850-851 area), add `text = region.get("text", "").strip()` at the start of the loop body, before the `if not text or self._is_noise(text):` check.

### File: `backend/services/vlm_structure_mapper.py`
### Lines to change (~line 850-852):

```python
        for i, region in enumerate(sorted_regions):
```
